# Student Expense Tracker

A professional expense tracking web application built using **Python, Streamlit, SQLite, Pandas, and Plotly**.

This application helps users manage their monthly budgets, track expenses, analyze spending patterns, and monitor their financial habits through an interactive dashboard.

---

## Features

### User Authentication

* User Registration
* Secure Login System
* Individual user accounts

### Budget Management

* Set monthly spending budget
* Automatic monthly budget reset
* Track remaining budget

### Dashboard

* Total Budget
* Total Expenses
* Remaining Budget
* Highest Spending Category

### Expense Management

* Add expenses
* Categorize spending
* Add notes
* Record expense dates

### Expense History

* View all expenses
* Delete expenses
* Export expense reports as CSV

### Analytics

* Category-wise spending analysis
* Interactive Pie Chart
* Interactive Bar Chart
* Monthly spending insights

### Currency Converter

* Supports:

  * INR
  * USD
  * EUR
  * GBP
  * JPY
  * KWD

## Installation

Clone the repository:

git clone https://github.com/jotshna/Student-Expense-Tracker.git

Navigate to the project folder:
cd Student-Expense-Tracker
Install dependencies:
pip install -r requirements.txt
Run the application:
streamlit run app.py
